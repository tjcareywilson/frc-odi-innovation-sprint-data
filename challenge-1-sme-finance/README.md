# Challenge 1: Structured data for SME finance and investment

**The question:** What changes if Companies House data is treated as a reusable, trusted foundation for SME finance and investment decisions?

## Files included

| File | Description |
|------|-------------|
| `Accounts_Bulk_Data-2025-12-25.csv` | Companies House accounts bulk data (~39k rows) |
| `UK_gender_pay_gap_data_2024_to_2025.csv` | Full UK Gender Pay Gap data 2024-25 (~16.5k employers) |
| `charity_commission_register_sample.tsv` | 2,000-row sample of the Charity Commission register |
| `BoE_Bankstats_TableA8.1_SME_lending.xls` | Bank of England Table A8.1: monthly SME lending |
| `BoE_Bankstats_TableA8.1.1.xls` | Bankstats Table A8.1.1: additional SME lending breakdowns |
| `BoE_Bankstats_TableA8.1.2.xls` | Bankstats Table A8.1.2: further SME lending detail |
